Linear Mechanisms

Now that you have the game elements inside of your robot from your intake, you need to be able to get them to the location where they need to be scored. In most FTC games, your drivetrain alone will not be able to do this. Why? In most modern FTC games, there is a vertical component to scoring That is to say, you're going to need a way to lift the game pieces up. While this can be accomplished via shooting in certain niche circumstances, I don't have the knowledge neccesary to create an intro guide to shooting in FTC. In most games, you will end up using a linear mechanism to combat this vertical challenge.
